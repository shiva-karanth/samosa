# SAMOSA Website

This is a simple, responsive website about Samosa, a popular South Asian snack. The site includes:

- Home page with history, nutrition facts, and fun facts about samosas
- Step-by-step guide on how to make samosas
- Ingredient and calorie breakdown
- Maintenance page for other cuisines
- FAQ section

## Folder Structure
- `index.html` — Main landing page
- `make.html` — How to make samosas
- `maintain.html` — Maintenance/other cuisines
- `faq.html` — Frequently asked questions
- `resources/` — All images and icons used on the site
- `style.css`, `make.css` — Custom stylesheets

## How to Run
Just open `index.html` in your browser. No build steps or dependencies required.
